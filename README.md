## **2. Portfolio Intelligence Dashboard**

- Hierarchical treemap

## **11. Machine Learning Advisor**
A Random Forest model trained on 5 years of data:

- SMA ratios

- **BUY / AVOID signal**  
- Probability score  
- CV metrics (Accuracy, AUC, Precision, Recall)
